This plugin allows the admin to select add a widget to their sidebar that shows the number of days since a specified date. One day at a time.

== Description ==

This plugin adds a counter widget to the WordPress sidebar area.

For more information visit:

[WP Days Since Widget](http://justinrains.com/)

== Installation ==

1. Upload the `wp-date-counter` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and you should see the Sobriety Counter widget
4. Enter the text that is displayed above and below the number of days.
5. Click Save.
6. View your site.
